ABOUT

Hey! This is my small project which simulates a browser games' website.
I've built it with the help of:

  • Angular 9
  • Angular Material
  • RxJS
  • REST API
  • GraphQL
  • Apollo Client (GQL for Angular)
  • Lots of energy drinks (did it all in less than 24h of work, special thanks here)
  • Some hope

As for a small quickstart for this website's navigation:

  1. The Homepage contains a list of ALL games available within the service.
    There is also a searchbar (triggered on submit) to ease Your burden while looking for anything specific.
  2. The Categories page contains a list of available categories from which You can select the one that suits You best,
    which then redirects You to the chosen category page with all it's games.
  3. The games' images contain an attached URL by default, however as it's a simulation -
    the URLs are empty, thus once clicked on an image - the browser will open a new tab to the homepage.
  4. The Mobile page is implemented with the use of GraphQL through Apollo Client.
  5. There is Routing implemented with params to determine correct category combined with routerLinks in navbar.
    (You can also type ex. localhost:port/category/popular-games directly into the browser)
  6. The games are listed with the use of mat-action-list, but as there are some missing resources -
    in some cases the arrangement might slightly differ due to the missing images.
